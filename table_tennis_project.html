Table tennis aerodynamics

Github page: Table tennis aerodynamics

This project envolves three of my passions: table tennis, physics, and coding. As a table tennis player for more than 15 years, I have some intuition about how the spin works, and building this simulation was fun and helped me understand the theory of something I have always approached using my "feeling".
The animations all have the same initial velocity, but different spin, and showcase how the trajectories are affected.
